Verification and Validation:

Talk about testing of different subsystems. How do we know it meets the requirements? Throughout the development and build stages we have been keeping a test plan which details each of our tests and which requirements they verify.

Limitations:

The robot has been tested in a controlled environment.

The robot must generate its own map of the environment before being used for intended purpose.

The robot relies on visual odometry to perform.

The robot can navigate corridors with the assumption that doors would be automatic.

One drawer is fully functional with scalability for other compartments available in the software.

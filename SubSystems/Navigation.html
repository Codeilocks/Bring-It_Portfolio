Motor Control

We can use keyboard presses to manually control the robot and this is useful for maintanence and allowing the robot to map the hospital rooms. Takes goal and is able to travel to waypoints. Uses arduino to drive motors. Pi tells arduino the cmd_vel or the key_press. THe cmd_vel and key_press is converted to microseconds the motors read and allow it to move in whatever direction is required.

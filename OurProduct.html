What we have achieved

Security: We have a lock mechenism whereby the draw will only be unclocked when the correct RFID token is read by the scanner

Automation (Draws): Through use of a linear actuator, once the draw has been unlocked it will be pushed partly open for the user to finish opening the rest

User Interface: We have a professional looking user interface which is simple to use

Task Manager: We have a system in place which communicates which deals with the orders and communicates with the robot

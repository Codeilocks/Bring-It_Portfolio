The Autonomous Delivery Robot MARC

MARC: Medical Assistant for Retrieval and Collection

Our robot is designed to be used by hospital staff to order and retrieve equipment from an intuitive user interface. MARC can autonomously navigate comfortably through hospital corridors, alongside patient beds and other people, utilising environment mapping and obstacle avoidance.

What we have achieved

Security: We have a lock mechanism whereby the drawer will only be unlocked when the correct RFID token is read by the scanner.

Automation (Drawers): Once the drawer has been unlocked, it will be pushed partly open by a linear actuator for the user to finish opening the rest of the way.

Navigation: Utilising level 3 automation, the robot is able to travel to given goals in pre defined settings, and can also be manually controlled by a keyboard.

User Interface: We have a professional looking user interface which is simple to use for requesting equipment.

Task Manager: We have a system in place which deals with the orders and communicates with the robot.

Fault Diagnosis: The battery levels are read correctly and the robot will be notified when battery level is low.
